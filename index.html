Cocina Descriptive Metadata

The following sunburst diagram is an attempt to visualize the shape of Cocina descriptive metadata for objects in the Stanford Digital Repository. Cocina is expressed using JSON, and each wedge represents a portion of a JSON path for used in assertion, and is sized based on how many assertions it is part of. The JSON path radiates from the center out, and wedges can be clicked on to reveal additional parts of a path. The data was generated using the descriptive shape report which generated a CSV file.
